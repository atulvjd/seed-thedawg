# Open Source Quadruped Robot

## Vision
An affordable open-source quadruped robot built using off-the-shelf components that anyone can recreate.

## Objectives
- Fully documented build process
- Modular hardware and software
- Beginner friendly
- Affordable components
- Open-source ecosystem

## Planned Features
- Walking gait
- Remote control
- Stable balancing
- Modular leg design
- Expandable sensor support

## Hardware Plan
- ESP32 / Raspberry Pi
- Servo motors
- Li-ion battery
- 3D printed frame
