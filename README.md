# Open Source Quadruped Robot

An open-source quadruped robot project built using off-the-shelf and easy-to-find components. This repository documents the complete journey of designing, building, testing, and improving a robo dog from scratch. The goal of this project is to create a modular, affordable, and beginner-friendly robotics platform that anyone can learn from, replicate, modify, and expand upon.

This repository will include complete hardware documentation, electronics, wiring diagrams, CAD models, firmware, software, testing logs, design iterations, failures, improvements, research, and development progress throughout the entire engineering process. Every major step of the project will be documented clearly so students, makers, robotics enthusiasts, developers, and researchers can understand how the robot is being built from the ground up.

The project focuses heavily on transparency and reproducibility instead of only showing final polished results. Failed prototypes, unstable designs, servo issues, balance problems, optimization attempts, and real-world engineering challenges will also be documented as part of the development journey.

The long-term vision is to create a fully documented and expandable open-source quadruped robotics platform that can serve as:
- A learning resource for robotics beginners
- A reference project for makers and developers
- A base platform for AI and robotics experiments
- An educational robotics system
- A community-driven open-source robotics project

---

## Project Goals

- Build a fully functional quadruped robot from scratch
- Use affordable and accessible components
- Create modular hardware and software architecture
- Document every stage of development in detail
- Make robotics easier to understand for beginners
- Share code, CAD, and engineering resources openly
- Develop stable movement and walking systems
- Create scalable foundations for future upgrades

---

## Planned Features

- Quadruped walking system
- Modular leg mechanism
- Servo-based movement architecture
- Remote control functionality
- Stable balancing system
- Expandable sensor support
- Open-source firmware and software
- Real-time testing and debugging
- Future AI and computer vision support

## Repository Structure

docs/          → Documentation, research, roadmap, and build logs  
hardware/      → CAD files, STL files, wiring diagrams, and mechanical systems  
firmware/      → ESP32/microcontroller code and motor control systems  
software/      → Control systems, simulations, and future AI modules  
tests/         → Servo tests, gait testing, balance testing, and experiments  
media/         → Images, videos, renders, and project visuals  
BOM/           → Bill of materials and component references

## Development Philosophy

This project is being built with a strong focus on:
- Open-source engineering
- Transparency in development
- Learning through experimentation
- Detailed documentation
- Real-world robotics problem solving
- Community accessibility and reproducibility

Rather than hiding mistakes or failed designs, this repository will document the complete engineering process including both successes and failures to help others learn robotics more effectively.
